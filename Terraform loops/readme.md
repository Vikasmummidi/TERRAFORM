| Data Type  | Allowed Loop Variables | Is `index` Available? | Ordered? |
| ---------- | ---------------------- | --------------------- | -------- |
| List/Tuple | `index`, `value`       | ✅ Yes                 | ✅ Yes    |
| Map/Object | `key`, `value`         | ❌ No (`key` used)     | ✅ Yes    |
| Set        | `value` only           | ❌ No                  | ❌ No     |
